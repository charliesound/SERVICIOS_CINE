# SESION 11: Antigravity - Demo Comercial

## Rol
Senior Product Engineer

## Contexto
Disenar la demo comercial de SERVICIOS_CINE para mostrar a clientes potenciales.

## Objetivo
Crear una experiencia de demo que:
1. Sea memorable y impressive
2. Muestre capacidades en 5 minutos
3. No requiera preparacion previa
4. Deje claro el valor del producto

## Flujo de Demo Propuesto

### 1. Login Rapido (30s)
- Usar cuenta demo preconfigurada
- Mostrar dashboard con stats

### 2. Generar Imagen (1min)
- Elegir tipo: still
- Escribir prompt cinematico
- Ver resultado en segundos

### 3. Crear Video (1min)
- Elegir tipo: video
- Usar imagen de referencia
- Ver animacion

### 4. Probar Doblaje (1min)
- Elegir tipo: dubbing
- Seleccionar preset de voz
- Escuchar resultado

### 5. Ver Cola (30s)
- Mostrar trabajos en tiempo real
- Explicar priorizacion

### 6. Mostrar Planes (30s)
- Comparativa visual
- Enfatizar valor de Studio/Enterprise

## Demo Script

```
"Hola, hoy les voy a mostrar SERVICIOS_CINE, 
nuestra plataforma de servicios audiovisuales con IA.

Empezamos rapido - ya tengo una cuenta demo configurada.
(Ventana de login)

Aqui vemos el dashboard. Tenemos 4 backends activos:
still para imagenes, video para video, dubbing para audio...

Ahora voy a crear una imagen cinematica.
(Escribe prompt: 'robot en atardecer, estilo Blade Runner')

Nuestro sistema detecta automaticamente el mejor workflow
y genera la imagen en segundos.

(Slide de imagen generada)

Impresionante, verdad? Y esto es solo el plan Free.

Ahora probemos el plan Studio con video.
...
```

## Cuentas Demo Necesarias
- demo_free: para mostrar limites
- demo_studio: para mostrar capacidades completas
- demo_admin: para mostrar panel de control

## Parametros de Exito
- Duracion total: 5-7 minutos
- Tiempo hasta primer resultado: < 2 minutos
- Numero de "wows": minimo 3
- Claridad del mensaje de valor: alto
