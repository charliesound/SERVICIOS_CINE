# Solicitud de Material para Diagnóstico y Piloto

El material solicitado depende del problema detectado.

## Para diagnóstico
- Descripción del flujo actual
- Herramientas usadas
- Principales problemas
- Roles implicados

## Para piloto CID
- Guion
- Presupuesto si existe
- Storyboard si existe
- Reports
- Media ya ingestada si aplica
- Documentación de ayudas si aplica
- Contactos comerciales si se prueba CRM
- Restricciones de confidencialidad

## Nota operativa

AILinkCinema pide solo el material necesario para entender el cuello de botella y validar la solución adecuada. Si el problema no encaja con CID, el alcance del material puede cambiar.
