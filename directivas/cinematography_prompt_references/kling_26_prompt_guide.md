# Kling 2.6 Prompt Guide (Reference)

Core structure:
- Subject: who/what is in frame
- Action: what is happening now
- Environment: where it happens and key props
- Style: visual language and tone
- Camera: shot type, angle, lens, motion
- Emotion: intended audience feeling

Practical checklist:
- Keep one primary action per shot.
- Keep camera motion limited and intentional.
- Preserve character identity, wardrobe, and location continuity.
- Avoid contradictory lighting/color instructions.
