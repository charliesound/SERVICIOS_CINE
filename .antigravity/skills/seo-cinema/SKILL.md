# SEO Cinema

Use this skill when building public pages that should communicate clearly to search engines and humans in the cinema/audiovisual domain.

## SEO Rules

- Ensure a single clear `h1` per page.
- Provide concise page titles and meta descriptions when the stack allows it.
- Use section headings that match product and industry intent.
- Keep body copy specific to cinema and audiovisual workflows.

## Content Rules

- Mention terms like production, storyboard, dubbing, postproduction, delivery, and audiovisual pipeline where relevant.
- Avoid vague AI-only marketing language without domain context.
- Keep copy structured for scanability: short paragraphs, lists, cards, and section grouping.

## Technical Rules

- If SSR metadata is not available, at least set `document.title` and a description meta tag on public pages.
- Prefer semantic HTML structure with visible headings and descriptive link text.
- Do not sacrifice readability for keyword repetition.
