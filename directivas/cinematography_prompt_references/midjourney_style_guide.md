# Midjourney Style Guide (Reference)

Useful cinematic still components:
- Shot type and framing
- Camera angle and focal length
- Depth of field
- Film stock/look reference language
- Lighting style
- Color palette and grading
- Composition intent

Quality controls:
- Avoid overcrowded scenes.
- Keep one visual focal point per shot.
- Prioritize readability over decorative detail.
