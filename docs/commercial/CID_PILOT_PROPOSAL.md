# Propuesta de Diagnóstico + Piloto AILinkCinema / CID

## Objetivo general

Entender el problema real de la productora y validar si CID es la solución adecuada para mejorar la coordinación integral del proyecto audiovisual.

## FASE 1 - Diagnóstico AILinkCinema

### Objetivo
Entender el problema real de la productora.

### Incluye
- Entrevista de flujo de trabajo
- Detección de cuello de botella
- Análisis de herramientas actuales
- Propuesta de solución IA

### Resultado esperado
- Diagnóstico claro del problema operativo
- Decisión sobre si CID encaja o si hace falta otra solución

## FASE 2 - Piloto CID si aplica

### Objetivo
Validar CID con un proyecto real o controlado.

### Material
- Guion
- Presupuesto si existe
- Reports
- Documentación
- Carpeta de media ya ingestada si se prueba postproducción
- Necesidades comerciales / distribución

### Entregables
- Diagnóstico
- Dashboard CID
- Presupuesto orientativo
- Storyboard asistido si aplica
- Dossier productor
- Pack distribución
- CRM básico
- FCPXML DaVinci si aplica

## Qué se validará
- Si la coordinación entre roles mejora
- Si el tiempo de preparación de materiales baja
- Si el equipo trabaja con una base de información más clara
- Si la salida comercial y editorial gana consistencia

## Qué no entra
- Solicitudes oficiales de ayudas
- Distribución asegurada
- Montaje final
- Presupuesto final cerrado

## Criterios de éxito
- Identificación clara del cuello de botella
- Uso útil del dashboard y módulos relevantes
- Mejora percibida en coordinación y preparación
- Decisión informada sobre continuidad

## Siguiente paso
1. Reunión de diagnóstico AILinkCinema
2. Confirmación de alcance
3. Recogida de material
4. Piloto CID si encaja
