## 1. Título comercial privado

`Script-to-Production Breakdown — del guion a una primera lectura de producción`

Esta es una ficha privada para conversaciones cualificadas. No es una landing pública, no es una oferta comercial abierta y no presenta el sistema como producto final.

## 2. Problema que resuelve

En una fase inicial de producción es difícil pasar rápidamente de un guion a una visión organizada de:

- escenas;
- personajes;
- localizaciones;
- riesgos;
- viabilidad;
- presupuesto preliminar;
- puntos que requieren revisión humana.

El problema no es sustituir al productor, sino ayudar a ordenar una primera conversación profesional antes de invertir tiempo en presupuestos, reuniones o desgloses más profundos.

## 3. Qué hace la demo

La demo toma un guion ficticio controlado, `Proyecto Demo Bruma`, y genera una salida estructurada para conversación de producción.

Incluye:

- demo controlada;
- guion ficticio;
- desglose;
- viabilidad;
- riesgos;
- presupuesto preliminar revisable;
- Excel editable `.xlsx`.

## 6. Qué valor aporta en una primera reunión de producción

- ayuda a ordenar la conversación;
- detecta riesgos iniciales;
- ayuda a ver qué falta;
- permite discutir presupuesto preliminar sin venderlo como definitivo;
- facilita preguntas entre producción, dirección y dirección de producción;
- no sustituye el criterio profesional.
