# AILink Sync Dialogue — Commercial Readiness QA v1

## 1. Objetivo

Este documento revisa el estado comercial real de AILink Sync Dialogue antes de enseñar la herramienta a terceros, publicar una landing real o contactar escuelas, productoras y equipos de postproducción.

La finalidad es separar claramente:

- Qué se puede enseñar ya.
- Qué no se puede prometer.
- Qué material usar en una demo.
- Qué mensaje comercial usar.
- Qué falta antes de publicar.
- Qué falta antes de automatizar captación.
- Qué decisión conviene tomar ahora.

Esta fase no implementa frontend, backend, formulario, CRM, Supabase, pagos, vídeo, tracking, cookies, Docker, runtime, CID SaaS ni configuración.

## 2. Estado general

AILink Sync Dialogue está en estado de pack comercial pre-beta.

Existe material suficiente para enseñar una demo controlada y validar interés con contactos concretos, pero no existe todavía un producto público completo.

Estado correcto de comunicación:

- Herramienta local-first.
- Prototipo funcional controlado.
- Demo con metadata controlada.
- Beta privada futura.
- Flujo manual de leads.
- Sin prometer automatización completa.
- Sin prometer integración directa con editores.
- Sin pedir material sensible de clientes todavía.

## 4. Qué se puede enseñar ya

Se puede enseñar en una demo privada o conversación controlada:

- El concepto de AILink Sync Dialogue.
- La landing estática como borrador visual.
- El reporte HTML de ejemplo.
- La tabla de archivos detectados.
- La tabla de sugerencias de match.
- El enfoque local-first.
- La beta privada como fase limitada.
- El flujo de feedback manual.

- La separación entre AILink Sync Dialogue y CID.

Frase segura:

AILink Sync Dialogue ayuda a preparar vídeo, audio y metadata antes de entrar en montaje, generando salidas revisables para ordenar el material y detectar relaciones útiles entre archivos.

## 5. Qué no se puede prometer

No prometer:

- Producto público terminado.
- SaaS disponible.
- Acceso inmediato automático.
- Subida segura de material sensible.
- Formulario real funcionando.
- CRM funcionando.
- Supabase conectado a producción.
- Pago online activo.
- Instalador para Windows o macOS.
- Integración directa con DaVinci Resolve, Avid o Premiere.
- Sincronización por waveform.
- Transcripción.
- Reconocimiento visual de claqueta.
- Edición automática.
- Sustitución del montador.
- Resultados garantizados.

## 8. Qué post publicar primero

Publicar primero un post de validación, no de venta agresiva.

Mensaje recomendado:

AILink Sync Dialogue está preparando una beta privada para equipos audiovisuales que necesitan ordenar vídeo, audio y metadata antes de montaje. Buscamos conversaciones con escuelas de cine, productoras pequeñas y equipos de postproducción que quieran aportar feedback sobre un flujo local-first.

Objetivo del primer post:

- Medir interés.
- Conseguir conversaciones.
- Identificar problemas reales.
- No vender todavía como producto cerrado.

## 9. Qué decir a una escuela de cine

Mensaje recomendado:

Estamos preparando AILink Sync Dialogue, una herramienta local-first para ayudar a estudiantes y equipos docentes a revisar carpetas de rodaje, detectar vídeo y audio, leer metadata disponible y generar salidas revisables antes de montaje. Nos interesa validar si este flujo puede ahorrar tiempo en prácticas, cortos y ejercicios de postproducción.

Pregunta clave:

¿Tendría sentido probar una beta controlada con material de práctica o metadata simulada, sin pedir material sensible?

## 10. Qué decir a una productora

Mensaje recomendado:

Estamos preparando AILink Sync Dialogue para equipos que reciben carpetas de rodaje y necesitan ordenar vídeo, audio y metadata antes de entrar en montaje. La herramienta está pensada como apoyo local-first para revisar material, detectar relaciones útiles y generar informes exportables.

Pregunta clave:

¿Dónde se os pierde más tiempo ahora: identificando archivos, relacionando audio/vídeo, revisando metadata o preparando material para montaje?

Antes de contactar:

- Elegir 5 a 10 contactos de alta calidad.
- Preparar mensaje corto manual.
- Elegir imagen de apoyo.
- Abrir la landing estática y revisar visualmente.
- Revisar el reporte HTML de demo.
- Definir qué se pide exactamente al contacto.
- Preparar una respuesta si piden probar.
- Preparar una respuesta si preguntan por precio.
- Preparar una respuesta si preguntan por integración con editores.
- Preparar una respuesta si preguntan por privacidad.

## 14. Decisión recomendada ahora

Decisión recomendada:

No implementar todavía CRM, Supabase, formulario ni pagos.

Paso recomendado:

Contactar manualmente con 5 a 10 leads cualificados o preparar primero el vídeo demo si se quiere una pieza visual más fuerte.

Orden sugerido:

1. Revisar internamente este QA.
2. Elegir una imagen principal.
3. Elegir un post inicial.
4. Preparar una lista de contactos.
5. Hacer 5 contactos manuales.
6. Registrar respuestas.
7. Decidir si merece la pena implementar formulario/CRM.

## 15. Riesgos comerciales

Riesgos actuales:

- Prometer demasiado pronto.
- Confundir AILink Sync Dialogue con CID.
- Presentar la demo como producto final.
- Entrar en infraestructura antes de tener señales de interés.
- Pedir material sensible sin marco legal cerrado.
- Hablar de integración con editores antes de tenerla.
- Hablar de IA como sustitución del montador.
- Publicar demasiado sin tener flujo de respuesta preparado.

## 16. Señales de validación positiva

Señales positivas:

- Una escuela acepta una demo.
- Una productora describe un problema real de orden de material.
- Un montador pide probar con un caso controlado.
- Un DIT ve valor en metadata/reportes.
- Un equipo pregunta por precio beta.
- Alguien pide recibir novedades.
- Un contacto acepta una segunda conversación.

## 17. Señales de que hay que ajustar

Señales de ajuste:

- No se entiende qué hace la herramienta.
- Se confunde con edición automática.
- Se confunde con CID completo.
- El público pide primero transcripción.
- El público pide primero integración con DaVinci/Avid/Premiere.
- El público no ve valor sin waveform sync.
- El mensaje local-first no queda claro.
- La landing parece demasiado técnica.

## 18. Próxima acción recomendada

Próxima acción recomendada:

Preparar una lista manual de 10 leads cualificados y un mensaje corto de contacto para escuelas de cine y productoras.

No abrir fase técnica de CRM/Supabase/formulario hasta tener señales reales.
