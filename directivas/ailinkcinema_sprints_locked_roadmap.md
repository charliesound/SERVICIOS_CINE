# AILinkCinema: Locked Roadmap (Sprints 1-6)

| Sprint | Propósito | Valor para Demo | Valor para Venta | Exclusiones |
| :--- | :--- | :--- | :--- | :--- |
| **1** | **Foundation Build** | UI Premium, Dashboard de Stats, Login. | Demuestra seriedad de producto y capacidad técnica de infra. | Ingesta real de guion, lógica de renders complejos. |
| **2** | **Narrative Ingestion** | Creación de Proyectos, Escenas y carga de Script. | Posiciona la plataforma como "Audiovisual" y no solo "AI Tool". | Parseo automático de guion (manual base). |
| **3** | **Visual Engine V1** | Renders Still/Video/Audio integrados en el flujo de escenas. | Prueba de que los modelos de IA se orquestan profesionalmente. | Edición/Sequencer (solo renders aislados). |
| **4** | **Traceability Hub** | Visión de "Coste por Escena" y "Activos vinculados". | El valor diferencial: control de presupuesto y rastro de activos. | Integración con software contable externo. |
| **5** | **Review & Collab** | Portal de Cliente (external) con sistema de notas/feedback. | Cierra el ciclo de servicio: el cliente del productor ya ve resultados. | Chat en vivo, edición colaborativa simultánea. |
| **6** | **Producer Area Final** | Dashboard ejecutivo completo + Módulo "Funding Matching". | El gancho comercial final: ayuda a financiar el proyecto. | Pasarela de pago real (solo leads y matching manual). |

## Definiciones de Cierre

### Foundation Build (S1)
Infraestructura de red, seguridad básica, sistema de carpetas industriales y UI base consistente.

### Internal Demo (S2-S3)
El equipo puede ejecutar un flujo completo de "Crear Proyecto -> Subir Idea -> Lanzar Render" aunque requiera comandos manuales.

### External Demo (S4-S5)
Un comercial puede mostrar la plataforma a una productora sin miedo a fallos de flujo. El valor de "Trazabilidad" es visible.

### Sale-Ready (S6)
El producto tiene "Feature Parity" con la promesa comercial. Listos para los 5 primeros pilotos de pago.

---
**Regla de Oro:** Ningún sprint se considera cerrado sin el "Checklist de Validación Premium" completado.
