# AILinkCinema Landing

Use this skill when editing public-facing landing pages for AILinkCinema or CID.

## Intent

- Position AILinkCinema as the parent brand and technology partner.
- Position CID as the flagship product.
- Present specialized apps as separate but connected solutions.

## Design Rules

- Prefer a dark premium visual system with cinematic gradients and restrained glow.
- Keep sections spacious, readable, and commercially focused.
- Avoid dashboard-like density, noisy layouts, and long copy blocks.
- Use large type, short supporting text, and clear CTA hierarchy.
- Make the output feel closer to premium cinema SaaS than generic startup landing pages.

## Messaging Rules

- Lead with business clarity before feature detail.
- Use audiovisual vocabulary: production, pipeline, storyboard, dubbing, post, delivery.
- Keep copy concise and confident.
- Make brand hierarchy explicit: AILinkCinema -> CID -> specialized apps.

## Implementation Notes

- Reuse shared landing components when they already fit the visual language.
- Prefer responsive grids, cards, and section-based composition.
- Keep changes isolated to frontend marketing surfaces unless explicitly requested otherwise.
